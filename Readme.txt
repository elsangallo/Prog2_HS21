Semesterarbeit Elias Pfändler: Banic
Banic ist ein Webapp indem Konten verwaltet werden können. 
Es stehen vier Konten zur Verfügung, das Lohnkonto, das Sparkonto, das Mietkonto und das Haushaltskonto.
Bei diesen können Transaktionen vorgenommen werden, so kann beispielsweise Geld transferiert, abgehoben oder eingezahlt werden. 
Ebenfalls kann aus dem Startscreen der aktuelle Kontostand abgerufen werden.
